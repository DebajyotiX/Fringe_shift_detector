# Fringe_shift_detector
It detects fringes' positions in a CCD image and calculates the shift between two sets of fringes.

RUN--> fringe shift measurinator.py

Required libraries
1. matplotlib.pyplot as plt
2. numpy
3. cv2
4. scipy
